Initial setup for React with Typescript projects.

Ready to deploy.

## Installation

Clone the repository, then run:

```bash
npm install
```

## Usage

### Development mode

```bash
npm start
```

### Generate build

```bash
npm run build
```

### Serve build

```bash
npx webpack serve
```

or

```bash
serve -s dist
```
